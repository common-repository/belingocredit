== Description ==

The plugin allows you to place a loan calculator on your page
<a href="https://belingo.ru/kreditnyj-kalkulyator/">View a loan calculator demo</a>

== Installation ==

1. Upload `belingoCredit` to the `/wp-content/plugins/` directory.
2. Activate `belingoCredit` in your wordpress site if it is not activated already.
3. Place shortcode [belingoCredit] on any page.

== Frequently Asked Questions ==

= How to display the calculator on the page? =

Just place a shortcode [belingoCredit]
